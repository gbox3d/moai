# 모아이 리시버


## 명령어
apinfo   
원격디바이스의 주소를 확인한다.  
동글을 처음 시동할경우 이명령어를 수행하여 원격디바이스의 주소를 확인한다.   


activate [시간]   
지정한 시간동안 전송상태를 활성화 한다.  
```bash
activate 10000
```
10초간 전송상태를 활성화 한다.  


## 응답형식

'#' 시작 : 시스템 메시지  
'%' 시작 : 씨리얼 통신 응답 데이터 메시지   

'$' 시작 : 센서 데이터 메시지   

발사 카운트, 발사모드샐랙트, 베터리 잔량, quaterion w, x, y, z  

```txt
$ 6 1 5.043000 0.972961 -0.000977 0.004944 0.230652
```
